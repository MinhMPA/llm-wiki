# Advanced Workflows

## Subagent-Driven Synthesis Drafting

When a host supports background agents, a background agent may transform a query/answer exchange into a filing candidate.

The background agent may prepare:

- target page path
- draft page body
- source-record citations
- index entry
- log entry
- validation notes

The background agent must not write durable wiki pages without human approval. In v1 this is an advanced reference workflow, not a required behavior. Future plugin packaging may expose it as a separate bundled skill.
