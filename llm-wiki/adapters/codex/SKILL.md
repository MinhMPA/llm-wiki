# LLM Wiki Adapter For ChatGPT Codex

This adapter is thin. Shared workflow logic lives in `../../core/`.

Before operating on an LLM Wiki:

1. Read `../../core/README.md`.
2. Read `../../core/references/workflows.md`.
3. Read `../../core/references/record-contracts.md`.
4. Read `../../core/references/page-contracts.md`.
5. For validation behavior, read `../../core/references/validation.md`.

When working inside a starter wiki, read `WIKI_SCHEMA.md` first. `AGENTS.md` is only a pointer file; it is not the source of truth.
